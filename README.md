# ALEX AI

Website portfolio hub for ALEX AI.

This static site showcases websites built by ALEX AI and includes:

- A hero section with generated website-preview artwork
- A filterable work gallery
- Services and process sections
- A contact form that opens an email draft

## Edit Projects

Update the `projects` array and `contactEmail` value in `app.js` to add real website names, descriptions, and links.

## GitHub Pages

This site is ready to serve from the repository root on the `main` branch.
